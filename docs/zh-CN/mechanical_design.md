# 机械设计 — Hsiwu

## 设计理念

Hsiwu 机械结构围绕三个原则设计：

1. **仿生保真度** — 分段翅膀运动学必须忠实地再现真实蝙蝠（以中华鼠耳蝠 *Rhinolophus luctus* 为形态学参考）翼膜的耦合扑动-伸展运动。
2. **可制造性** — 所有结构部件均可在标准消费级 FDM 3D 打印机上无需支撑打印。
3. **模块化** — 子装配体各自独立；损坏的翅膀段落或连杆可单独更换，无需重印整个机器人。

## 运动学基础

### 曲柄摇杆四连杆机构

将电机连续旋转转化为翅膀往复扑动的核心机构是**曲柄摇杆四连杆机构**（格拉霍夫类型：曲柄摇杆）。

```
                         连杆（rod_bc）
                    ┌─────────────────────────────┐
                    │                             │
    曲柄（rod_a）──┤                             ├── 摇杆（rod_d）
    （固定在电机     │                             │   （输出至翅膀段落）
     轴上，360° CW  │                             │
     连续旋转）      │                             │
                    │      机架（固定连杆）          │
                    └─────────────────────────────┘
```

**格拉霍夫条件：** 曲柄摇杆存在的条件是，最短杆与最长杆长度之和小于其余两杆长度之和：
```
s + l < p + q
```
其中 s = 最短杆，l = 最长杆，p、q 为其余两杆。

在 Hsiwu 连杆机构中：
- **曲柄**（rod_a）= 最短杆 — 由电机驱动，连续旋转
- **连杆**（rod_bc）= 中间件 — 将运动传递给摇杆
- **摇杆**（rod_d）= 输出件 — 以约 70° 幅度摆动
- **机架** = 机械框架（电机轴与摇杆枢轴之间的距离）

### 扑动角度

摇杆角行程（扑动角度）由连杆几何决定：
```
θ_max = arcsin((r_曲柄 + r_连杆) / r_摇杆)  [近似上界]
θ_min = arcsin((r_曲柄 - r_连杆) / r_摇杆)  [近似下界]
扑动幅度 = θ_max − θ_min ≈ 70°
```

### 六种连杆类型

每个翅膀段落（每侧3段）使用各自的连杆组。三组共享相同的运动学拓扑结构，但尺寸可能略有差异，以适应翅膀渐缩的平面形状。

| 连杆 | 角色 | 备注 |
|-----|------|-------|
| `rod_a` | 曲柄 | 连接电机输出轴；最短杆 |
| `rod_bc` | 连杆 | 连接曲柄与摇杆；BC 表示共享通用几何的两种变体 |
| `rod_d` | 摇杆 | 输出连杆；驱动翅膀段落 |
| `rod_h` | 辅助连杆 | 连接丝杠驱动螺母 |
| `rod_ij` | 剪式连杆 | 剪式伸缩机构的一部分 |
| `rod_kf` | 连接杆 | 翅膀段落之间的结构连接件 |

## 丝杠与剪式机构

### 设计目的

真实蝙蝠的翼膜不仅扑动，还会**伸展**——翼面弯度在扑翼周期中动态变化。Hsiwu 通过**丝杠驱动剪式连杆机构**复现此特性，在扑动过程中协调地扩展和收缩有效翼面积。

### 机构

```
   电机 → 2转3联轴器 → 丝杠（旋转）
                          │
                 移动螺母（直线平移）
                          │
                 剪式连杆（伸缩）
                          │
                 翼膜面积变化
```

### 丝杠规格

| 参数 | 数值 |
|-----------|-------|
| 螺距（P） | 0.4 mm |
| 有效直径（d₂） | 1.74 mm |
| 驱动力矩（M） | 0.0064 N·mm（计算值） |
| 减速比（电机:丝杠） | 1:4 |

### 力矩计算

参考文献（齐津浩、张卫平，2020）给出丝杠驱动力矩计算式：
```
M = F × d₂/2 × tan(α + ρ)
```
其中：
- F = 螺母轴向载荷
- d₂ = 有效直径
- α = 导程角 = arctan(P / (π·d₂))
- ρ = 摩擦角 = arctan(μ)

取 P = 0.4 mm，d₂ = 1.74 mm，μ ≈ 0.1（钢对塑料）：
```
α = arctan(0.4 / (π × 1.74)) ≈ 4.2°
ρ = arctan(0.1) ≈ 5.7°
M = 0.0064 N·mm
```

### 力学分析

剪式机构以位移换取力。螺母前进 Δx 时，剪式尖端横向位移为 Δy。机械优势为：
```
MA = F_y / F_x ≈ cot(β) / 2
```
其中 β 为剪式半角。剪刀接近闭合时（β → 小），MA 高；接近完全伸展时（β → 大），MA 降低。

## 结构框架

### 中央支撑（`central_support`）

主机身底盘。特征：
- 电机安装座（每侧三个）
- 丝杠轴承座
- PCB 安装柱（轮廓匹配）
- 框架装配用 M2/M3 通孔
- 减重镂空

### 头部支撑与尾部支架

- **头部支撑** — 固定前缘翅膀枢轴和丝杠前端轴承
- **尾部支架** — 固定后缘枢轴点

### 轴承与销钉

- **轴承** — 微型滚珠轴承在两端支撑丝杠，安装在 `center_bearing_cap` 和 `head_bearing_cover` 中
- **头销 / 尾销** — 翅膀段落铰接的精密铰链销

## 电机与联轴器

### 空心杯直流电机

参考电机型号为 ZWPD006006-26，空心杯（无铁芯）直流电机，配四级行星齿轮箱。

**空心杯电机优点：**
- 转子惯量更低 → 方向反转响应更快
- 无齿槽转矩 → 低速运行平稳
- 效率更高 → 电池续航更长

**四级行星齿轮箱：**
- 紧凑形体下实现高减速比
- 输入输出同轴 → 简化机械布局
- 扭矩均匀分配至各行星轮 → 耐久性好

### 2转3轴联轴器

将电机输出轴（直径 2 mm）连接到丝杠输入端（直径 3 mm）。联轴器容纳微量不对中，并通过紧定螺钉传递扭矩。

## 制造注意事项

### FDM 3D 打印

| 设置 | 数值 |
|---------|-------|
| 层高 | 0.2 mm |
| 喷嘴直径 | 0.4 mm |
| 填充 | 20–30%（螺旋或网格） |
| 外壳圈数 | 3 |
| 支撑 | 无需 |
| 材料 | ABS（耐用）或 PLA+（易打印） |

### 激光切割（连杆）

连杆可采用激光切割亚克力或胶合板，替代 3D 打印方案，提供更好的尺寸精度：
- 2–3 mm 厚亚克力板
- STL 文件位于 `mechanical/output/laser/`
- 切割后：去毛刺，试配轴承

### 装配提示

1. **先装轴承** — 在组装框架前将微型轴承压入其安装座
2. **预装配** — 不装螺丝先试装，验证所有间隙
3. **螺纹锁固** — 电机安装螺丝使用微量螺纹锁固剂（低强度，如 Loctite 222）
4. **连杆对齐** — 在连接电机之前验证所有四连杆机构在全行程范围内自由运动
5. **布线** — 电机导线沿框架内部通道走线至 PCB XH-2A 连接器
