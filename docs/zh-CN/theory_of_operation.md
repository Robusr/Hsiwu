# 工作原理 — Hsiwu 电子系统

## 概述

Hsiwu 控制系统采用纯模拟与数字逻辑元件（无微控制器、无固件、无可编程逻辑），产生三路独立同步的往复电机驱动信号。

## 系统框图

```
                   +-----------+
  3.7V 锂电池 ────│ 电源管理   │──── 7.35V ────────────── 电机轨（L293D VCC2）
                   │           │
                   │           │──── 5.0V ───┬────────── 逻辑轨（NE555, 74HC04, L293D VCC1）
                   +-----------+             │
                                             │
              ┌──────────────────────────────┤
              │                              │
       ┌──────┴──────┐               ┌──────┴──────┐
       │ NE555 (U1)  │               │ NE555 (U2)  │  ...
       │ 多谐振荡模式  │               │ 多谐振荡模式  │
       │ ~4.8 Hz     │               │ ~4.8 Hz     │
       └──────┬──────┘               └──────┬──────┘
              │ OUT1                        │ OUT2
       ┌──────┴──────────────────────┐     │
       │       74HC04 (U5)           │     │
       │  ┌─ 非门 ─ /OUT1            │     │
       │  ├─ 非门 ─ /OUT2            │◄────┘
       │  └─ 非门 ─ /OUT3            │
       └──────┬──────────────────────┘
              │
    ┌─────────┼─────────┐
    │   OUT1  │  /OUT1  │
    ▼         ▼         ▼
  ┌──────────────┐
  │ L293D (U6)   │──── 电机1（前缘段）
  │ 半桥驱动      │
  └──────────────┘
  （U7/电机2/中段、U8/电机3/后缘段同理）
```

## 第一级：电源供应

### 输入保护
- **D1**（肖特基二极管，SMC封装） — 防反接保护。电池反接时D1阻断电流。
- **SW2**（SS12D07VG4滑动开关） — 总电源开关。
- **LED1**（绿色0805 LED） — 电源指示，串联限流电阻。

### 升压转换器（MT3608，U4）
MT3608 为定频（1.2 MHz）电流模式升压转换器。

- **输入**：3.7 V（单节锂电池，标称值）
- **输出**：7.35 V（由反馈电阻分压器设定）
- **电感**（L1）：SMD屏蔽功率电感，7.3×6.8 mm
- **输入电容**：10 µF（C1，0805 X7R或X5R）
- **输出电容**：22 µF（C3，SMD铝电解等效）

输出电压由下式给出：
```
V_OUT = V_REF × (1 + R1/R2)
```
其中 V_REF ≈ 0.6 V（MT3608内部基准），R1、R2构成反馈分压网络。

### LDO 稳压器
- 线性稳压器将 7.35 V 升压输出降至干净的 5.0 V，供逻辑 IC（NE555、74HC04）和 L293D 逻辑电源（VCC1）使用。
- 去耦电容（C2、C4）靠近 IC 电源引脚放置。

## 第二级：时序发生（NE555 多谐振荡器）

每片 NE555（U1、U2、U3）配置为**多谐振荡模式**。
在此模式下，输出在 HIGH（≈ VCC）与 LOW（≈ 0 V）之间持续振荡，无需外部触发。

### 电路配置

```
           VCC
            │
           ─┼─ R_A ──┬── R_B ──┬── VCC
            │        │         │
            │        ├── 引脚7（DISCH）
            │        │
            ├────────┼── 引脚6,2（THRES, TRIG）
            │        │
            │       ─┼─ C
            │        │
           ─┴────────┴── GND
```

### 时序方程

**输出高电平时间（t_H）：**
```
t_H = 0.693 × (R_A + R_B) × C
```

**输出低电平时间（t_L）：**
```
t_L = 0.693 × R_B × C
```

**周期：**
```
T = t_H + t_L = 0.693 × (R_A + 2·R_B) × C
```

**频率：**
```
f = 1/T = 1.44 / ((R_A + 2·R_B) × C)
```

### 设计参数（~4.8 Hz）

| 参数 | 数值 | 备注 |
|-----------|-------|-------|
| R_A | ~10 kΩ（固定+微调） | U1为 R + R4（微调） |
| R_B | ~100 kΩ | 固定 0805 电阻 |
| C | ~1 µF | 陶瓷或钽电容 |
| **目标频率 f** | ~4.8 Hz | 参考设计10 Hz按比例降低 |
| **目标周期 T** | ~208 ms | |
| **占空比** | ~52% | 设计上略有不对称 |

**频率微调：** 每路 NE555 有一个可调电阻（U1为R4，U2为R10，U3为R11）——三端 SMD 微调电位器。调节这些可改变 R_A，从而改变振荡频率，实现各段时序独立校准。

### 启动同步

三片 NE555 共享同一 VCC 轨，RC 网络相近，上电时趋向于近似同步启动。但由于元件公差，会逐渐漂移。无论相对相位如何，74HC04 级确保每个电机始终获得互补驱动信号。

## 第三级：相位反相（74HC04）

74HC04 为六反相器（六个独立非门）。仅使用三个：

| 门 | 输入 | 输出 | 用途 |
|------|-------|--------|---------|
| 1A → 1Y | NE555 U1 OUT | /OUT1 | L293D U6 反向驱动 |
| 2A → 2Y | NE555 U2 OUT | /OUT2 | L293D U7 反向驱动 |
| 3A → 3Y | NE555 U3 OUT | /OUT3 | L293D U8 反向驱动 |

每路 NE555 输出同时连接至对应 L293D 输入引脚**和**一个 74HC04 输入。反相输出连接至互补 L293D 输入引脚。

### 信号关系

```
NE555 OUT = HIGH → L293D IN1 = HIGH, IN2 = LOW  → 电机正转
NE555 OUT = LOW  → L293D IN1 = LOW,  IN2 = HIGH → 电机反转
```

以此实现连续正反转往复运动。

### 未使用的门

三路未使用的反相器（4A/4Y、5A/5Y、6A/6Y）的输入端应接 GND，防止浮空输入振荡和额外电流消耗。

## 第四级：电机驱动（L293D）

L293D 为双 H 桥驱动芯片，每通道可提供 600 mA（峰值 1.2 A）。

### 引脚配置（每电机，使用一个半桥）

| L293D 引脚 | 连接 | 用途 |
|-----------|------------|---------|
| VCC1（引脚16） | 5 V 轨 | 逻辑供电 |
| VCC2（引脚8） | 7.35 V 轨 | 电机供电 |
| 1,2EN（引脚1） | 5 V（常使能） | 半桥1-2使能 |
| 1A（引脚2） | NE555 OUT | 输入A — 方向控制 |
| 2A（引脚7） | 74HC04 /OUT | 输入B — 互补方向 |
| 1Y（引脚3） | 电机端子+ | 电机输出 |
| 2Y（引脚6） | 电机端子− | 电机输出 |
| GND（引脚4,5,12,13） | 公共地 | 功率+逻辑回流 |

### 真值表

| 1A (IN1) | 2A (IN2) | 电机 |
|----------|----------|-------|
| H | L | 正转（电机两端+7.35 V） |
| L | H | 反转（电机两端−7.35 V） |
| L | L | 制动（电机两端短接至GND） |
| H | H | 制动（电机两端短接至VCC2） |

正常工作时，1A 和 2A 始终互补，电机始终处于正转或反转驱动状态——不会制动。制动状态仅在逻辑切换的几纳秒过渡期间出现。

### 内置保护

L293D 内含：
- **续流二极管**位于所有输出端——无需外加二极管
- **热关断**——防止过热
- **过流限制**——内部限流

## 系统级时序

一个翅膀段落的完整信号链：

```
t = 0:    NE555 OUT 变 HIGH
          → L293D IN1 = HIGH, IN2 = LOW
          → 电机正转
          → 翅膀段落上扑

t = t_H:  NE555 OUT 变 LOW（约108 ms后，~52%占空比）
          → L293D IN1 = LOW, IN2 = HIGH
          → 电机反转
          → 翅膀段落下扑（回程）

t = T:    NE555 OUT 再次变 HIGH（约208 ms总计）
          → 循环以 ~4.8 Hz 重复
```

## 设计理由

### 为什么选用 NE555？

- **简洁** — 三个外部元件（R_A、R_B、C）即可设定频率
- **宽供电范围** — 4.5 V 至 16 V，兼容 5 V 逻辑轨
- **驱动能力充足** — 可输出/吸入 200 mA，足以驱动 74HC04 输入和 L293D 逻辑输入
- **久经考验** — NE555 是史上文档最完备的 IC 之一

### 为什么选用 74HC04，而非直接从 NE555 取互补输出？

NE555 输出级为图腾柱（推挽）结构——仅提供单一输出。互补输出需外部晶体管反相器或逻辑门。74HC04 单颗芯片提供六个反相器，三路 NE555 通道仅用三个——简洁紧凑。

### 为什么是 ~4.8 Hz（而非参考文献的 ~10 Hz）？

参考样机（齐津浩、张卫平，2020）实现约 10 Hz 扑动频率。Hsiwu 设计目标为约 4.8 Hz——约为一半频率——原因如下：

1. **更大单次扑动幅度** — 较低频率使翅膀在空气阻尼截止前达到完整 70° 行程
2. **降低机械应力** — 3D 打印连杆和框架承受更低循环载荷
3. **可观察性** — 在 4.8 Hz 下，分段翼膜变形肉眼清晰可辨
4. **电机电流余量** — 较低频率意味单节锂电池平均电流消耗更低

通过调节三路微调电位器（R4、R10、R11）减小 R_A，可在系统机械极限范围内提高频率。
